# Modules List

This document summarizes the main software modules for the AI Mock Interview & Tutor Booking Marketplace.
It is derived from the source idea in [AI_Mock_interviews.md](../AI_Mock_interviews.md) and the full API catalog in [all_apilist.md](all_apilist.md).

## 1. Core Product Domains

- Identity and access management
- Tutor marketplace and discovery
- Scheduling and availability
- Booking and payment flow
- Live interview session delivery
- Notifications and reminders
- Wallet, commission, and withdrawals
- AI interview analysis and scoring
- Organization hiring programs
- Reporting, audit, and support operations
- File handling, storage, and platform services

## 7. Notes

- The API catalog currently defines 40 implementation modules.
- Several of the source-idea features are represented as sub-features inside these modules.
- The frontend should consume these modules through the backend API only.
- The backend should remain API-first while the frontend stays in the separate frontend folder.
