# User Stories

## Student Stories

- As a student, I want to search tutors by category, price, rating, and time so that I can find a suitable interview coach.
- As a student, I want to book a slot and pay securely so that my session is confirmed immediately.
- As a student, I want reminders before the interview so that I do not miss the session.
- As a student, I want feedback after the session so that I can improve.

## Tutor Stories

- As a tutor, I want to create availability slots so that students can book me.
- As a tutor, I want to accept or manage booking changes so that my calendar stays accurate.
- As a tutor, I want to see earnings and withdrawals so that I can track my income.
- As a tutor, I want to view learner performance so that I can improve coaching quality.

## Organization Stories

- As an organization, I want to onboard candidates in bulk so that I can manage interview programs efficiently.
- As an organization, I want reporting dashboards so that I can measure outcomes.
- As an organization, I want role segregation so that recruiters and admins have different permissions.

## Admin Stories

- As an admin, I want to manage user verification so that platform quality stays high.
- As an admin, I want to inspect payment and dispute records so that I can resolve incidents.
- As an admin, I want to configure categories and notification templates so that operations remain consistent.

## Enterprise Stories

- As a super admin, I want organization-level configuration and audit logs so that governance is enforceable.
- As a super admin, I want platform analytics so that I can make capacity and growth decisions.
